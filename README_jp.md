OpenRAGは、インテリジェントなドキュメント検索とAIを活用した会話を可能にする、総合的なRetrieval-Augmented Generation（RAG）プラットフォームです。ユーザーは大規模言語モデルとセマンティック検索機能を備えたチャットインターフェースを通じて、ドキュメントのアップロード・処理・クエリを行うことができます。本システムはドキュメントの取り込み、検索ワークフロー、およびインテリジェントな提案にLangflowを活用し、シームレスなRAGエクスペリエンスを提供します。[Starlette](https://github.com/Kludex/starlette)と[Next.js](https://github.com/vercel/next.js)で構築されており、[OpenSearch](https://github.com/opensearch-project/OpenSearch)、[Langflow](https://github.com/langflow-ai/langflow)、[Docling](https://github.com/docling-project/docling)を基盤としています。
