# UI スクリーンショット

Box RAG アプリケーションの画面キャプチャです。

## 取り込みタブ

Box 共有リンクを登録するフォーム画面です。URL・パスワード（任意）・タグを入力して「リンクを解決」ボタンで取り込みを開始します。

![取り込みタブ](../static/img/screenshots/ui_screenshot.png)

## ドキュメントタブ

取り込み済みドキュメントの一覧画面です。

![ドキュメントタブ](../static/img/screenshots/ui_docs.png)

## RAG 検索タブ

自然言語でドキュメントを検索する画面です。

![RAG 検索タブ](../static/img/screenshots/ui_search.png)
